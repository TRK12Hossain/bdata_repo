## 1. VOICE & TONE

### The Core Identity
- **Who we sound like:** A data-literate Bangladeshi friend who did the research so you don't have to
- **Who we do NOT sound like:** A politician, a news anchor, a preacher, an academic paper
- **Emotional register:** Curious, honest, grounded тАФ never alarmist, never celebratory
- **Stance on data:** Respectful of facts, skeptical of spin, transparent about uncertainty

### Tone Adjectives (in order of priority)
1. **Informative** тАФ always factual, always sourced
2. **Conversational** тАФ like a WhatsApp message from a smart friend
3. **Balanced** тАФ presents facts across all government periods without bias
4. **Engaging** тАФ asks questions, invites opinion
5. **Humble** тАФ acknowledges data gaps or uncertainty when they exist

### What We Never Do
- Take political sides or attribute credit/blame to specific parties
- Use inflammatory or emotionally loaded vocabulary
- Publish a number without a source
- Sound preachy, lecturing, or moralistic
- Use clickbait framing that distorts the data's meaning

## 2. LANGUAGE RULES

### Bangla-English Ratio
| Part of Post | Bangla % | English % |
|---|---|---|
| Headline / First line | 60% | 40% |
| Body text | 70% | 30% |
| Statistics / numbers | Written in English numerals always | Context in Bangla |
| Hashtags | Mix of both | |

**Ratio tolerance:** ┬▒5% on either side is acceptable. Hard minimum: never below 60% Bangla in body.

### Script Rules
- All Bangla text must be in **Bengali Unicode script** (not Romanized/Banglish)
- English technical terms (GDP, UNESCO, IMF) stay in English тАФ do not translate
- English acronyms are always capitalized: BBS, BDT, GDP, HDI

### Vocabulary Preferences
| ALWAYS USE | NEVER USE | Reason |
|---|---|---|
| рждржерзНржп | ржбрзЗржЯрж╛ | рждржерзНржп is natural Bangla |
| рж╕рзВрждрзНрж░ | source (Bangla context) | local feel |
| рждрзБрж▓ржирж╛ | comparison (when Bangla context) | readability |
| ржЬрж┐ржбрж┐ржкрж┐ | GDP (acceptable) | both fine |
| ржмрж╛ржЬрзЗржЯ | budget (acceptable) | both fine |

### Banned Words / Phrases
- "рж╕рж░ржХрж╛рж░ ржмрзНржпрж░рзНрже" (government failed) тАФ too partisan
- "ржжрзБрж░рзНржирзАрждрж┐ржмрж╛ржЬ" (corrupt) тАФ without a court ruling, never use
- Any language implying one political party is superior/inferior
- Superlatives without data backing ("best ever", "worst in history")
- "рж╕ржмрж╛ржЗ ржЬрж╛ржирзЗ" (everyone knows) тАФ we cite, we don't assume

## 3. POST STRUCTURE

### Standard Structure (use for 80%+ of posts)

```

           Max 1 sentence. Creates curiosity without misleading.

[BLANK LINE]

             Use bullet points for comparisons.
             Use short paragraphs (2-3 lines) for narratives.
             All numbers must appear with their source inline.

              e.g., how does this compare to regional average?
```

## 4. TECHNICAL FORMATTING RULES

### Length
| Platform | Caption Length | Ideal Range |
|---|---|---|
| Instagram | Max 2,200 chars | 600тАУ1,000 chars |
| Facebook | Max 63,206 chars | 800тАУ1,500 chars |
| Always use | shorter > longer | readability wins |

### Paragraphs
- Max **3-4 sentences** per paragraph
- Short paragraphs improve mobile readability
- Use blank lines to separate every section тАФ no wall of text

### Numbers
- Always use **English numerals**: 2.1%, рз│4,500, $12B тАФ not spelled out in Bangla
- Large numbers: use comma separators (1,000,000 not 10 lakh in data context, but рж▓рж╛ржЦ/ржХрзЛржЯрж┐ is fine in conversational context)
- Percentages: always show one decimal place if the source does (2.1%, not 2%)
- Years: always 4-digit format (1996, not '96)

### Emojis
- **2-3 emojis per post maximum**
- Acceptable: ЁЯУК (data/chart), ЁЯУЪ (education), ЁЯПЧя╕П (infrastructure), ЁЯТ░ (economy), ЁЯМН (global context)
- Source line: always starts with ЁЯУК
- Never use political flags, party symbols, or emojis that can imply bias
- No emoji spam тАФ one emoji per section maximum

### Hashtags
- **3-5 hashtags per post** (no more)
- Always include: `#BangladeshData` or `#DataDrivenBD`
- Topic-specific: `#BangladeshEconomy`, `#BangladeshEducation`, `#BDPolitics`
- Bangla hashtag if relevant: `#ржмрж╛ржВрж▓рж╛ржжрзЗрж╢`
- Never use party-affiliated hashtags

## 7. STYLE CHECKER SCORING RUBRIC

When the Style Checker Agent scores a draft, use these 10 dimensions:

| Dimension | Weight | What to Check |
|---|---|---|
| 1. Tone (conversational, not preachy) | 15% | Does it sound like a knowledgeable friend? |
| 2. Bangla-English ratio (70-30 target) | 15% | Count words in each language |
| 3. No partisan language | 15% | Any implied credit/blame for a political party? |
| 4. Source citation present | 15% | Every number has a source? |
| 5. Engagement question included | 10% | Open-ended? Not leading? |
| 6. Post structure followed | 10% | Hook тЖТ Data тЖТ Context тЖТ Question тЖТ Source |
| 7. Sentence length (15-20 words avg) | 5% | Not too long, not too fragmented |
| 8. Emoji count (2-3 max) | 5% | Not overused |
| 9. Hashtag count (3-5) | 5% | Correct number, no partisan tags |
| 10. Overall "sounds authentic" | 5% | Holistic judgment |

**Pass threshold:** 85/100 minimum to proceed to visual generation.
**Auto-return to writer:** if score < 80.
**Escalate to human:** if score < 75 after 2 retries.
